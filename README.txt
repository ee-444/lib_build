This project is used to build archives from object files using ar.exe.  Look in each folder for the .bat files for details.

2 archive builds. Includes object files from 022 release. Path ArduinoLib is the core files from Arduino.cc and 
RoverLib includes the all the object files from ArduinoLib and also the object files for peripherals that the 
rover needs.
